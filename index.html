Arduino (Jeenode) dual channel temperature monitor

I put together this project so I could monitor the temperature of my lean-to green house I made to protect some plants from the frost this winter.

Parts

The project uses a Jeenode with a RFM12B 868MHz transceiver, a DS18B20 temperature sensor (inside lean-to) and a probe version of the DS18B20 (outside temp.) The display is a 128x64 pixel mono Graphical LCD. Both transmitter and receiver use a 1500 mah lithium battery cell to power so both use powersaving code to extend battery life.

Details

The transmitter sends out the two temp's and battery level every 60 seconds sleeping in between transmissions, the receiver waits for the first transmission then updates the display with the temp's and remote battery level then updates the local battery level than sleeps for 53 seconds before waking up for the next transmission.

NOTE. to get the battery levels both tansmitter and receiver must be powered straight to VCC and not via the voltage regulator otherwise the voltage would stay at 3.3V until the battery died.
